# Dang Good Pies

An over-engineered bakery website built with React and Vite. Features 3D tilt effects, smooth page transitions, animated pie charts, parallax scrolling, and a picnic-themed UI.

## Features

- **3D tilt cards** — pie cards respond to mouse movement with perspective transforms
- **Smooth page transitions** — fade and slide animations between sections
- **Animated deliciousness metrics** — pie charts that count up on load
- **Horizontal scroll showcase** — featured pies with smooth scroll behavior
- **Picnic theme** — red/white diagonal pattern, warm color palette throughout
- **Responsive layout** — hamburger nav with full-height dropdown on mobile
- **Multi-page structure** — Home, Menu, Our Story, Contact

## Pages

| Page | Content |
|------|---------|
| Home | Featured pies, specials section, mission statement |
| Menu | Full pie catalog with descriptions and pricing |
| Our Story | Bakery background and values |
| Contact | Location, hours, order info |

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Lucide React (icons)

## Getting Started

```bash
# Install dependencies
npm install

# Run dev server
npm run dev

# Build for production
npm run build
```
